# SimpleMD User Test Scenarios

## S1. 앱 실행 — Welcome 화면
- [ ] 앱 실행 시 Welcome 마크다운이 편집 모드로 열림
- [ ] 상태바에 "Editing" 표시
- [ ] 커서가 보임
- [ ] 기본 테마가 GitHub 스타일

## S2. Finder에서 .md 파일 더블클릭
- [ ] .md 파일 더블클릭 시 SimpleMD으로 열림
- [ ] 뷰어 모드(Read Only)로 열림
- [ ] 커서가 안 보임
- [ ] 마크다운 문법(#, **, ``` 등)이 숨겨지고 렌더링됨

## S3. CLI로 파일 열기
- [ ] `simplemd view file.md` — 뷰어 모드로 열림
- [ ] `simplemd edit file.md` — 편집 모드로 열림
- [ ] `simplemd file.md` — 기본(편집) 모드로 열림

## S4. 뷰어 모드 동작
- [ ] 클릭해도 커서 안 보임
- [ ] 텍스트 선택 하이라이트 안 됨
- [ ] 활성 라인 하이라이트 안 됨
- [ ] 키보드 입력해도 편집 안 됨
- [ ] 상태바에 "Read Only" 표시

- [ ] 전환 시 스크롤 위치 유지
- [ ] 편집 모드에서 상태바 "Editing" 표시

## S6. 코드 블럭 렌더링
- [ ] 뷰어 모드: ``` 마커 숨겨지고 스타일링된 코드 블럭으로 표시

- [ ] 편집 모드에서 커서가 코드 블럭 밖에 있으면 렌더링된 코드 블럭 표시
- [ ] 다양한 언어 태그 지원 (```js, ```python 등)

## S8. 수학 수식 (LaTeX)

## S10. 인라인 마크다운 렌더링

- [ ] `inline code` — 코드 스타일, ` 마커 숨김
- [ ] [link](url) — 링크 스타일, 문법 숨김
